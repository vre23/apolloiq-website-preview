Automating Pathology Filing

Taunton Vale Healthcare, a busy Somerset practice serving over 14,000 patients, faced growing pressure to manage thousands of routine blood tests. Filing normal results was consuming valuable clinician time daily -- time that should've been spent on patient care.

That's where ApolloIQ stepped in.

The Solution

ApolloIQ deployed a pathology automation system that processes 80+ different biomarker tests across 17 categories, capable of automatically filing 70-80% of blood test results that fall within normal ranges. The system uses gender-specific and age-specific reference ranges, clinically reviewed and validated, to ensure every result is assessed with precision.

It also includes sample quality detection, identifying when samples are haemolysed, insufficient, or otherwise unavailable -- preventing false results from being filed.

The system applies intelligent actions for each result: File Without Comments for normal results, Skip for results that need clinician review, and Sample unavailable when quality issues are detected -- allowing clinicians to focus only on the results that matter most.

17 Test Categories

Full Blood Count (FBC) Liver Function (LFTs) Urea & Electrolytes Lipid Profiles Inflammatory Markers Thyroid Function Bone Profile Vitamin D, B12, Folate HbA1c (Diabetes) Cardiac Biomarkers PSA (Age-based) And more...

How It Works

  • Reports received in EMIS format from pathology labs
  • Values extracted using pattern matching against known test structures
  • Each value compared against clinically reviewed reference ranges
  • Gender and age-specific thresholds applied automatically
  • Normal results filed automatically; abnormal flagged for doctor review
  • All processing logged for a full audit trail
